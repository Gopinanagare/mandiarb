# Mandiarb - Global Omni-Parity Arbitrage Intelligence

Mandiarb is an advanced, high-performance arbitrage terminal designed to identify and execute cross-market price parities across Agriculture, Automotive, Metals, and Retail sectors in India.

## Features
- **Real-Time Spread Engine:** Instantly tracks price variations between different physical and digital markets.
- **Premium Paywall System:** Locks highly profitable, low-risk routes behind a Razorpay subscription wall.
- **Google Authentication:** Seamless one-click login for secure access via `@react-oauth/google`.
- **Deep Search & Analytics:** Omni-channel filtering by category, buy city, and sell city with a dedicated analytics breakdown per trade.
- **Optimized UI:** Professional-grade Material Design 3 UI with automated data blurring for unauthenticated users.

## Tech Stack
- React 19 + Vite
- Tailwind CSS
- React Router DOM
- Razorpay Payments Integration
- Google OAuth 2.0

## Getting Started
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.
